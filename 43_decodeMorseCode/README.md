# Decode Morse Code



# Problem Description:

Write a function called decodeMorseCode which takes a string of Morse Code as an argument and returns a string of decoded Morse Code.

# Input

A string

## Output

A string

### Examples:

#### Input:

```js
decodeMorse('.... . .-.. .-.. --- / .-- --- .-. .-.. -..');
```

#### Output:

```js
'HELLO WORLD'
```

#### Input

```js
decodeMorse('.. -. -.. .. ...- .. -.. ..- .- .-.. ... / .- -. -.. / .. -. - . .-. .- -.-. - .. --- -. ... / --- ...- . .-. / .--. .-. --- -.-. . ... ... . ... / .- -. -.. / - --- --- .-.. ...')
```

#### Output

```js
'INDIVIDUALS AND INTERACTIONS OVER PROCESSES AND TOOLS'
```
